HTML Text Formatting

HTML text formatting refers to the use of specific HTML tags to modify the appearance and structure of text on a webpage. It allows you to style text in different ways,
such as making it bold, italic, underlined, highlighted, or struck-through.

HTML contains several elements for defining text with a special meaning.

Emphasized: This text is emphasized and italic.

Bold: This text is bold.

Italic: This text is italic.

Marked: This text is highlighted.

Underlined: This text is underlined for emphasis.
